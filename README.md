# Password Strength Tester

You can test strength of your password

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Install dependencies

```bash
  npm install
```

Start

```bash
  npm  start
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
